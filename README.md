# ETL Automatizado com Python

Projeto simples de ETL desenvolvido para prática de Engenharia de Dados.

## Objetivo
Ler um dataset bruto de vendas, aplicar transformações e gerar um arquivo tratado.

## Transformações realizadas
- Tratamento de valores nulos
- Conversão de datas
- Criação de coluna de ano

## Tecnologias utilizadas
- Python
- Pandas

## Estrutura
- data/raw → dados brutos
- data/processed → dados tratados
- scripts → lógica do ETL#
